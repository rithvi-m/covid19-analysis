🦠 COVID-19 Data Analysis

Exploratory data analysis on COVID-19 global dataset to uncover 
trends in daily cases, deaths, and recovery rates.

 📊 Charts
- Daily confirmed cases trend
- Death rate analysis
- Recovery rate over time

 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

🔍 Key Insights
- Identified peak waves in case growth
- Analysed death-to-recovery ratio trends
- Visualised country-wise spread patterns

 📁 Dataset
COVID-19 public dataset — global records

*Project completed as part of Data Science Internship at Kodbud*
